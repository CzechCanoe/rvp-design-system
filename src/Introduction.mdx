# CSK RVP Design System

Design systém pro registrační a výsledkový servis **Českého svazu kanoistů**.

## Účel

Tento design systém slouží jako sjednocující knihovna pro veřejné a komunitní sekce webových stránek:

- **Kalendář závodů** - přehled a filtrování závodů
- **Výsledkový portál** - archiv výsledků
- **Live výsledky** - real-time sledování závodů
- **Přihlašovací systém** - registrace na závody
- **Veřejné profily** - závodníci a kluby
- **Správa členství** - členská základna

## Design principy

- **Mobile-first** - optimalizace pro mobilní zařízení
- **WCAG 2.1 AA** - přístupnost pro všechny uživatele
- **Dual personality** - utilitární pro backoffice, expressive pro veřejné části
- **Dark/Light mode** - podpora obou barevných režimů

## Technický stack

- React 18+ s TypeScript (strict mode)
- Vite pro build
- Storybook 8 pro dokumentaci
