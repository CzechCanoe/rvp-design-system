# CSK RVP Design System

Design systém pro registrační a výsledkový portál **Českého svazu kanoistů**.

## Co zde najdete

### Prototypes (celé stránky)

Ukázky kompletních stránek - kalendář, výsledky, profily závodníků atd.

**Každý prototyp má dvě varianty:**

- **Embed** - Komponenta vložená do stránky kanoe.cz (bez vlastního headeru)
- **Satellite** - Samostatná aplikace s vlastním headerem (např. zavody.kanoe.cz)

### Components (jednotlivé prvky)

Tlačítka, karty, tabulky, formuláře a další stavební bloky.

## Ovládání v toolbaru

Vpravo nahoře najdete přepínače:

- **Theme** - Světlý / Tmavý režim
- **Mode** - Utility (kompaktní) / Expressive (prostornější) / Embed (pro kanoe.cz)

**Poznámka k tmavému režimu:** Pro Embed varianty používejte vždy světlý režim. Tmavý režim je určen pouze pro budoucí samostatné aplikace.

## Jak začít

1. Klikněte na **Prototypes** v levém menu - uvidíte ukázky celých stránek
2. Prozkoumejte **Components** - jednotlivé UI prvky s dokumentací
3. Použijte přepínač Embed/Satellite pro zobrazení různých variant

## Technické info

- React 18+ s TypeScript
- CSS custom properties (tokeny)
- Podpora light/dark theme
- Mobile-first design
